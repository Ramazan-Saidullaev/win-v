# Установка и настройка приложения истории буфера обмена

## Быстрый старт

### 1. Установка системных зависимостей

```bash
sudo apt-get update
sudo apt-get install xclip xsel python3-tk
```

**Примечание:** `python3-tk` необходим для работы графического интерфейса приложения.

### 2. Установка Python зависимостей (если еще не установлены)

```bash
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Настройка прав доступа для горячих клавиш

Для работы горячих клавиш на Linux может потребоваться:

**Вариант 1: Добавление пользователя в группу input (рекомендуется)**
```bash
sudo usermod -a -G input $USER
```
После этого нужно **перелогиниться** в системе.

**Вариант 2: Запуск с sudo (для тестирования)**
```bash
sudo venv/bin/python3 clipboard_history.py
```

### 4. Запуск приложения

```bash
source venv/bin/activate
python3 clipboard_history.py
```

### 5. Настройка автозапуска

Чтобы приложение запускалось автоматически при включении ноутбука:

```bash
./setup_autostart.sh
```

## Использование

- Нажмите **Ctrl+Alt+Z** для открытия истории
- Используйте поиск для быстрого поиска по истории
- Нажмите **Enter** или сделайте двойной клик для вставки выбранного элемента
- Нажмите **Escape** для закрытия окна

## Отключение автозапуска

Если нужно отключить автозапуск:

```bash
rm ~/.config/autostart/clipboard-history.desktop
```

## Где хранится история?

История сохраняется в файл: `~/.clipboard_history.json`

## Проблемы?

### Горячие клавиши не работают

1. Убедитесь, что вы добавили себя в группу `input` и перелогинились
2. Или запустите приложение с `sudo` (не рекомендуется для постоянного использования)

### Буфер обмена не отслеживается

Убедитесь, что установлены `xclip` или `xsel`:
```bash
sudo apt-get install xclip xsel
```
